> 관련 문서: 전투기획문서.md / 시스템 기획.md

---

## 1. 시스템 개요

캐릭터는 경영 단계(공방)에서 관리되고, 탐사 단계에서 자동전투 방식으로 싸운다.
공방에서 탐사 파티를 구성하고, 전투 씬에서 초상화 클릭으로 스킬을 발동한다.

> ❌ 3×3 격자 배치 UI는 삭제됨.
> ❌ Tab_Squad (탐사대 탭) 은 삭제됨.
> ✅ 전투는 크로노 아크 참고 초상화 자동전투 방식.

---

## 2. 직업

| 직업 | 전투 특성 | 처치 보상 | 영입 방법 | 구현 |
|------|-----------|-----------|-----------|------|
| 낚시꾼 (Fisher) | 전투 특화. 공격력↑ 피해감소 | 없음 | 화폐로 영입 | ✅ 생성 |
| 잠수부 (Diver) | 피해 증가 | 아군 처치 시 아이템 | 화폐로 영입 | ✅ 생성 |
| AI | 피해 증가 | 아군 처치 시 화폐 | 고철로 제작 | ✅ 생성 |

## 3. 스탯

| 스탯 | 설명 | 최솟값 | 구현 |
|------|------|--------|------|
| 체력 | HP (1스탯 = HP 10) | 1 | ✅ |
| 건강 | 라운드/탐사 종료 시 자연 회복량 | 0 | ✅ (회복 로직 미구현) |
| 공격 | 공격력, 스킬 피해에도 영향 | 1 | ✅ |
| 민첩 | 공격속도 (min 5, 10당 공속 1) | 5 | ✅ |
| 행운 | 크리티컬 확률 +1%/행운, 회피율 +0.1%/행운 | 0 | ✅ |

### 치명타 규칙

- 크리티컬 발동 시 공격력 200% 피해
- 행운이 100 초과 시 → 초과 수치 1당 공격력 +1

## 5. 스킬 & 패시브 풀

`Data_Skills.js` / `Data_Passives.js` 에서 실제 효과 정의.
`CharacterManager.createCharacter()` 에서 Cog 등급 기준 랜덤 부여.

### 패시브 풀 (공격 범위 결정)

| Cog | 패시브 풀 |
|-----|-----------|
| 1 | 윗칸 타격, 앞칸 타격 |
| 2 | 앞칸 타격, 현재 칸 타격 |
| 3 | 현재 칸 타격, 대각 타격, 윗칸 타격 |
| 4 | 전열 전체 타격, 대각 타격, 앞칸 타격 |
| 5 | 전열 전체 타격, 현재 칸 타격, 후열 타격 |
| 6 | 전/후열 동시 타격, 후열 타격, 전열 전체 타격 |
| 7+ | 전체 칸 타격, 전/후열 동시 타격 |

### 스킬 풀

| Cog | 스킬 풀 |
|-----|---------|
| 1 | 기본 일격, 빠른 찌르기 |
| 2 | 연속 타격, 방어 자세 |
| 3 | 강타, 회피 기동, 독 도포 |
| 4 | 광역 타격, 강화 독, 순간 가속 |
| 5 | 폭발 타격, 전방 스캔, 철갑 관통 |
| 6 | 심해 압박, 전기 충격, 철벽 방어 |
| 7+ | 코어 오버로드, 심연의 포효 |

### 게이지 시스템

| 스킬 등급 | 요구 게이지 |
|-----------|-------------|
| 약 | 100 |
| 중 | 200 |
| 강 | 300 |

- 게이지 = 타격 횟수 + 피격 횟수 합산
- 라운드 종료 시 유지 / 탐사 종료 시 초기화

## 6. 기록칩 시스템

- 일러스트(얼굴 ID)에 귀속되는 영구 누적 기록
- 판을 넘어 유지. 같은 얼굴이 재등장해도 기록은 쌓임

저장 구조 (`nr_record_chips` 키):

```js
{
  "spriteKey": {          // 예: "char_0", "char_12"
    expeditions: 0,       // 총 탐사 동행 횟수
    kills: 0,             // 총 처치 이형 수
    deaths: 0,            // 총 사망 횟수
    highestRegion: "",    // 최고 도달 구역
    highestCog: 0,        // 역대 최고 코그
    veteran: false,       // 탐사 10회 이상 생존
    firstDay: 0,          // 첫 영입 Day
    deathLog: [           // 사망 연대기 (역순)
      { day, cog, round, killedBy }
    ]
  }
}
```

## 7. 생존 / 사망 처리

| 상태 | 조건 | 처리 |
|------|------|------|
| 전투불능 | 체력 0 | 라운드 종료 시 회수 가능 |
| 회수 | 라운드 종료 + 잠수정 생존 | 이후 전투 불가, 경영 단계에서 자원으로 회복 |
| 영구 사망 | 탐사 실패 (잠수정 파괴 or 전멸) | 캐릭터 데이터 삭제, 기록칩은 유지 |
