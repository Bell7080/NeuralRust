# 🖥️ UI 설계 — NEURAL RUST
> 크로노 아크 UI 구조 참고. 전 씬 레이아웃·컴포넌트 기준 문서.
> 최종 반영: 2026-03

---

## 레퍼런스: 크로노 아크에서 차용하는 것

| 크로노 아크 | Neural Rust 적용 |
|------------|-----------------|
| 상단 적 초상화 + HP 바 열 | 이형 일러스트 + HP 바 (전투 씬 상단) |
| 하단 아군 초상화 카드 배열 | 아군 캐릭터 초상화 + 게이지 + HP (전투 씬 하단) |
| 초상화 클릭 → 스킬 발동 | 게이지 충전 시 초상화 클릭 → 사용 스킬 발동 |
| 전투 로그 피드 | 중단 텍스트 피드 (공격·피격·스킬 로그) |
| 카드 획득 후 선택 UI | ExploreScene 슬롯 카드 선택 (이미 구현됨) |
| 상단 자원 HUD | AtelierHUD의 Day·Arc 표시 |

## PartyScene — 파티 편성 (신규 설계)

탐사 난이도(코그 상한)가 결정된 뒤 파티를 구성하는 화면.
탐사대 탭을 대체한다.

```
┌─────────────────────────────────────────────────────────────────┐
│  선택한 탐사 — 코그 N 이하                                       │
│ ─────────────────────────────────────────────────────────────── │
│                                                                  │
│   [ 파티 슬롯 — 최대 N명 ]                                       │
│   ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐  +                   │
│   │ 캐릭 │  │ 캐릭 │  │ 캐릭 │  │  빈  │                      │
│   │ 터 1 │  │ 터 2 │  │ 터 3 │  │ 슬롯 │                      │
│   └──────┘  └──────┘  └──────┘  └──────┘                       │
│                                                                  │
│ ─────────────────────────────────────────────────────────────── │
│   [ 캐릭터 목록 — 코그 상한 초과 캐릭터 비활성화 ]               │
│   [카드] [카드] [카드] [카드] [카드] ...  (스크롤)               │
│                                                                  │
│                     [  출  발  ]                                 │
└─────────────────────────────────────────────────────────────────┘
```

**핵심 규칙**:
- 코그 상한 초과 캐릭터는 선택 불가 (회색 처리)
- 파티 슬롯은 최소 1명 필요 (출발 버튼 활성화 조건)
- 선택 완료 → `nr_party` 저장 → BattleScene 진입

## BattleScene — 전투 (신규 설계, 크로노 아크 참고)

```
┌─────────────────────────────────────────────────────────────────┐
│  라운드 N / 5          잠수정 ██████████ HP                      │
│ ─────────────────────────────────────────────────────────────── │
│                                                                  │
│   [ 이형 영역 — 상단 1/3 ]                                       │
│   ┌──────────────┐    ┌──────────────┐                          │
│   │  이형 일러스트│    │  이형 일러스트│                          │
│   │  ████ HP 바  │    │  ████ HP 바  │                          │
│   └──────────────┘    └──────────────┘                          │
│                                                                  │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │
│   [ 전투 로그 피드 — 중단 1/4 ]                                  │
│   > 심해어의 공격! 김철수에게 12 피해                            │
│   > 이수정 빠른 찌르기 발동 — 심해어에게 18 피해                 │
│                                                                  │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │
│   [ 아군 초상화 영역 — 하단 1/3 ]                                │
│                                                                  │
│   ┌─────────┐   ┌─────────┐   ┌─────────┐                      │
│   │ 초상화  │   │ 초상화  │   │ 초상화  │                       │
│   │         │   │         │   │         │                       │
│   │ ▓▓ 게이지│   │ ░░ 게이지│   │ ████ 게이│                   │
│   │ ██ HP  │   │ ██ HP  │   │ ██ HP  │  ← 클릭 시 스킬 발동    │
│   │ 이름   │   │ 이름   │   │ 이름   │                          │
│   └─────────┘   └─────────┘   └─────────┘                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**인터랙션**:
- 게이지가 요구치 이상인 캐릭터 초상화 → 발광 + 클릭 가능
- 클릭 → 사용 스킬 발동 → 게이지 소모
- 전투 종료 → **RoundResultScene** (일시 결산)

**컴포넌트 분리 계획**:

| 컴포넌트 파일 | 역할 |
|--------------|------|
| `BattleScene.js` | 씬 메인, 전투 루프 제어 |
| `Battle_EnemyArea.js` | 이형 영역 렌더링 + HP 갱신 |
| `Battle_LogFeed.js` | 전투 로그 피드 텍스트 스크롤 |
| `Battle_PartyBar.js` | 아군 초상화 배열 + 게이지 + HP |
| `Battle_SkillOverlay.js` | 스킬 발동 연출 (플래시·텍스트) |

## RoundResultScene — 라운드 결산 (신규 설계)

```
┌─────────────────────────────────────────────────────────────────┐
│  라운드 N 결산                                                   │
│ ─────────────────────────────────────────────────────────────── │
│                                                                  │
│   획득 자원                                                      │
│   화폐 +N       고철 +N       아이템 N개                        │
│                                                                  │
│   캐릭터 상태                                                    │
│   [초상화]HP%  [초상화]HP%  [초상화]전투불능                    │
│                                                                  │
│   게이지 현황 (유지됨)                                           │
│   [게이지바]  [게이지바]  [게이지바]                            │
│                                                                  │
│             [ 다음 라운드 ]   [ 철수 ]                          │
└─────────────────────────────────────────────────────────────────┘
```

## ResultScene — 최종 결산 (탐사 종료 후)

```
┌─────────────────────────────────────────────────────────────────┐
│  탐사 종료 — Day N                                              │
│ ─────────────────────────────────────────────────────────────── │
│                                                                  │
│   총 획득 자원                                                   │
│   화폐 NNN   고철 NNN   아이템 N개   심해화폐 N               │
│                                                                  │
│   캐릭터 결과                                                    │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐                     │
│   │ 생존     │  │ 회수     │  │ ☠ 사망   │                     │
│   │ [초상화] │  │ [초상화] │  │ [초상화] │                     │
│   └──────────┘  └──────────┘  └──────────┘                     │
│                                                                  │
│   기록칩 업데이트 애니메이션                                     │
│   (탐사 동행 +1, 처치 +N, 사망 기록 추가 등)                   │
│                                                                  │
│                    [ 공방으로 돌아가기 ]                         │
└─────────────────────────────────────────────────────────────────┘
```

## 씬 전환 흐름

```
AtelierScene
    ↓  [탐색 버튼]
ExploreScene (코그 상한 선택)
    ↓  [카드 선택]
PartyScene (파티 편성)
    ↓  [출발]
BattleScene (라운드 1)
    ↓  [전투 종료]
RoundResultScene
    ↓  [다음 라운드] (최대 5)
BattleScene (라운드 N)
    ↓  [5라운드 완료 or 철수]
ResultScene (최종 결산)
    ↓  [공방으로]
AtelierScene
```

## 공통 UI 컨벤션

### 컬러 팔레트

| 이름 | HEX | 용도 |
|------|-----|------|
| BG | `#050407` | 베이스 배경 |
| ACCENT | `#a05018` | 주요 강조 (버튼 테두리, 게이지 충전) |
| BRIGHT | `#c8a070` | 밝은 강조 (클릭 가능 상태, 수치) |
| TEXT | `#c8bfb0` | 주 텍스트 |
| DANGER | `#a03018` | HP 위험·사망·경고 |
| GAUGE_FILL | `#c8a070` | 게이지 바 채움 |
| HP_FILL | `#40a060` | HP 바 채움 |
| HP_LOW | `#a03018` | HP 30% 이하 |

### HP 바 색상 규칙

| HP 비율 | 색상 |
|---------|------|
| 70% 이상 | `#40a060` (초록) |
| 30~70% | `#a0a020` (노랑) |
| 30% 미만 | `#a03018` (빨강) |

### 클릭 가능 상태 표시

- 게이지 충전된 초상화: BRIGHT 컬러 글로우 + 펄스 애니메이션
- 비활성(게이지 부족): 채도 낮춤 (alpha 0.5)
- 코그 상한 초과 캐릭터 (PartyScene): 전체 dim 처리

### 폰트 사용

`FontManager` 상수 사용 원칙 유지.

| 용도 | 사이즈 기준 |
|------|------------|
| 이형 이름 | `scaledFontSize(22)` |
| 캐릭터 이름 | `scaledFontSize(16)` |
| 수치 (HP·게이지) | `scaledFontSize(14)` |
| 전투 로그 | `scaledFontSize(13)` |
| 버튼 텍스트 | `scaledFontSize(18)` |
