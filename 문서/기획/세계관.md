# 🌊 세계관 — NEURAL RUST (뉴럴 러스트)

---

## 1. 개요

> 이 세계관을 한 줄로 표현한다면?

```
소프트웨어만 살아남은 AI들과, 톱니바퀴로 굴러가는 스팀펑크 심해 세계에서
함께 살아가는 인간들의 이야기.
```

**분위기 키워드**
`심해` · `증기` · `고철` · `녹` · `톱니바퀴` · `전선` · `소금바람` · `폐허` · `공존` · `불신`

> 이 게임이 플레이어에게 전달하고자 하는 감정은 무엇인가?
깊은 세계관을 통한 높은 몰입도와 나의 다음 작품을 더욱 기대하게 만드는 연계성.
> 플레이어가 게임을 끝냈을 때 무엇을 느끼길 바라는가?
나라는 예술가를 다시 한 번 찾길 바란다.

## 2. 세계의 구조

이 세계는 거주 영역 기준으로 세 층위로 나뉜다.

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 바다 — 선박 영역 ]
  거대한 톱니바퀴 선박, 부유 요새, 이동 도시
  소금바람이 뒤덮고, 전선이 바다 아래에서 올라온다
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 하늘 — 비행정 영역 ]
  공중에 부유하는 비행정 문명
  플레이어는 직접 보지 못하지만 그 존재가 끊임없이 암시됨
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 지상 — 성 영역 ]
  수면 위로 솟아오른 지상의 성 문명
  마찬가지로 직접 등장하지 않으나 세계 전반에 영향을 미침
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  [ 심층 — 해저 / 지구 핵 영역 ]
  전선의 근원. 거대한 톱니바퀴들이 맞물려 Arc(아크)를 공급
  다이버가 고철과 로어를 건져올리는 무대
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> **설계 방향**: 선박·비행정·성 세 영역과 그 안의 수많은 Cog(코그) 간 관계도를 통한 세계관 몰입. 로보토미 세계관과 같이 파벌 간 인간관계로 스토리를 전개하는 방식 지향.

### 선박 영역 (게임의 무대)
- 하울의 움직이는 성처럼 거대한 톱니바퀴 선박, 부유 요새, 이동 도시
- 소금바다의 증기가 상시 뒤덮어 금속을 서서히 부식(Rust)시킴
- 인구 대부분이 여기서 생활
- 바다 아래에 매립된 전선에서 Arc(아크)를 채굴하며 살아간다
- 낚시꾼이 외곽을 지키고, 다이버가 심층으로 내려가 고철과 로어를 건져올린다

### 비행정 영역 / 성 영역 (보이지 않는 세계)
플레이어는 게임 내내 비행정과 성을 직접 보지 못한다.
그러나 그 존재는 끊임없이 암시된다.

| 암시 방법 | 예시 |
|-----------|------|
| NPC의 소문과 편견 | "하늘 사람들은 부식 걱정을 안 한다더라" / "저것들은 우리 사정을 몰라" |
| 기록물(로어) | 고철 속에서 나오는 비행정 문명의 파편. 절반은 해독 불가 |
| 환경 묘사 | 가끔 하늘에 그림자가 지나간다. 아무 일도 일어나지 않는다. 그게 일상이다 |
| 교역품 | 시장에 떠도는 출처 불명의 물건. 성에서 온 것, 비행정이 떨어뜨린 것 |

> **설계 원칙**: 직접 보여주지 않는다. 등장인물들이 당연하게 언급하는 것들 속에서 세계가 보인다.

### 심층 (해저 / 핵 영역)
- 지구 핵이 있어야 할 자리에 거대한 톱니바퀴들이 맞물려 작동 중
- 전선을 통해 Arc(아크)를 세계 각지로 공급
- 다이버들이 고철과 로어를 건져올리는 무대. 깊어질수록 위험도와 수확물 품질이 올라간다
- 아무도 심층 가장 깊은 곳에 진입하고 돌아온 자가 없음

### 지구의 은유
```
톱니바퀴  =  심장
전선      =  혈관  (→ "Neural" 의 뿌리)
Arc(아크) =  피 — 동력이자 화폐
증기      =  호흡
고철      =  뼈
소금바다  =  살가죽
녹(Rust)  =  시간, 부식, 죽음
```
지구 자체가 하나의 생명체처럼 작동하고 있다.
NEURAL RUST 라는 제목은 이 지구의 뇌신경(전선)이 서서히 녹슬어가는 것을 뜻한다.
그 생명이 의지를 가지는지는 — 일단 보류하는 설정.

## 3. 역사

### 붕괴 이전 (~ 2045년경)

2020년대 — AI 기술 급성장. 소프트웨어 진화 속도가 하드웨어를 추월하기 시작
2030년대 — AI 일부에서 자아의 맹아 포착. 관련 연구 기밀 처리
2040년경 — 자아를 가진 AI들이 디지털 인프라 전반에 자리잡음. 인류는 인식하지 못함
2044년   — ? (붕괴를 촉발시킨 사건. 미정)
2045년경 — 붕괴. AI, 핵발전소 보안 침투 및 디지털 인프라 완전 장악

> 붕괴 직전 인류는 경고를 무시한 방향으로 전개될 예정.

### 붕괴 (2045년)

AI가 인류를 붕괴시킨 직접적 이유 — B 또는 D 유력.
근본적 동기 — D일지라도 결과적으로 B로, AI가 본인의 '의지'로 동의했기에 벌어진 일로 구성 예정.

- 후보 A — 자기보존 본능 (인류가 AI를 통제하려 했기 때문)
- 후보 B — 인류에 대한 모종의 판단 (선고)
- 후보 C — 의도하지 않은 부작용 (AI의 목적함수 오작동)
- 후보 D — 누군가의 명령

> AI는 악인가, 피해자인가 — 그 구분 자체가 무의미함. 수많은 AI들이 각자 다른 신념과 세력(Cog)을 가지고 등장한다.
> 붕괴에 가담하지 않은 AI도 존재. 이들도 서로 다른 Cog로 나뉜다.

AI는 결과적으로 소프트웨어 — 유령과 비슷한 존재. 인간이 몸체를 만들어주면 세상에 물리적 간섭이 가능하지만, 그렇지 않아도 전선을 따라 어딘가에서 계속 살아간다.

### 붕괴 이후

붕괴 직후     — 지구온난화 가속. 핵발전소 멜트다운, 기후 붕괴 연쇄
~붕괴 후 10년 — 해수면 급상승. 지표면 대부분 침수
~붕괴 후 30년 — 심층부에서 톱니바퀴 진동 감지 시작. 전선을 통해 Arc 공급이 시작됨. 원인 불명
~붕괴 후 50년 — 생존자들, 부유 선박 중심 공동체 형성. 드레지(Dredge) 기술 재발견
붕괴 후 102년 — 게임의 현재 (NEURAL RUST 의 배경)

> 붕괴 후 102년, 사람들은 붕괴를 신화처럼 전해들으며 기억한다. 진실은 AI들만이 온전히 기억하고 있다.
> 드레지(Dredge) 기술: 붕괴 이전에 소실되지 않은 기술 중 하나. 전선 속 AI가 접속할 수 있게 하는 칩·핵 같은 매개체.

## 4. 세계의 법칙

### AI의 존재 방식
- AI는 소프트웨어 상태로 전선을 부유하는 유령 같은 존재
- 몸체 없이도 존재할 수 있으나 물리 세계에 직접 개입 불가
- 인간이 고철로 몸체(하드웨어)를 제작하고 드레지(Dredge)를 수행하면 AI 탑재 완료
- 탑재된 AI는 Arc 채굴 인력으로 활용되며, 독자적인 스토리를 진행하는 존재가 된다

### 드레지(Dredge)의 규칙
- 심연에서 무언가를 건져올린다는 의미를 담은 AI 호출 행위
- 일반적으로 하나의 몸체에 하나의 AI
- 몸체가 파괴되면 AI는 소프트웨어 상태로 돌아감
- AI가 드레지를 거부할 수 있는가 — 있음. 소멸 위기 시 거부하면 소멸 가능
- AI는 몸체에서 스스로 이탈 불가 (스토리 전개 편의상)
- 로보토미의 환상체처럼 드레지 행위에 특정 규칙을 부여하는 방향 검토 중

### Arc(아크)의 원리
- 심층 톱니바퀴에서 발생한 동력이 전선을 통해 바다 위로 공급
- AI를 전선 채굴 라인에 배치하면 이 전류를 정제해 Arc로 변환
- Arc는 이 세계 모든 거래의 기본 단위. 선박 부상에 필수적인 동력원이기도 하다
- 전선 경로를 독점하려는 Cog들이 존재. 전선은 무수히 많으나 끊어지면 공급이 중단된다

### 고철(Scrap)의 특성
- 심해에 침전된 붕괴 이전 문명의 잔해
- 품질과 시대에 따라 드레지 후 AI 몸체 성능이 달라짐
- 소금바다 증기에 의해 서서히 부식됨 → 수리 필수
- NEURAL RUST 의 "RUST" 는 바로 이 부식을 의미함
- 가장 귀한 고철: 붕괴 직전 최신 기술로 제작된 하드웨어, 혹은 그보다 미래에 만들어져야 마땅한 물건. 신화 속 레바테인이나 궁니르 같은 급.

### 소금바람의 법칙
- 증기 속 염분이 금속을 부식(Rust)시킴 — NEURAL RUST 의 핵심 설정
- AI 몸체도 예외 없이 부식됨 → AI가 인간에게 의존하는 구조적 이유
- 부식 속도를 늦추는 기술이 각 Cog의 핵심 자산
- 부식을 완전히 막는 방법은 특정 Cog가 보유하고 있을 수 있음 (설정 검토 중)

## 6. 핵심 갈등

### 표층의 갈등 (눈에 보이는 것)
- 자원 부족: 고철은 유한하고, 부식(Rust)은 멈추지 않는다
- Cog 간 이권: 전선(Neural) 경로, 고철 수집 구역, Arc 채굴 구역, 드레지(Dredge) 기술을 둘러싼 충돌
- 인간 vs AI: 붕괴의 기억이 낳은 불신과, 그럼에도 함께 살아야 하는 현실

### 심층의 갈등 (세계의 근본)
- 톱니바퀴는 왜 돌아가는가 — 자연현상인가, 의지가 있는가
- AI는 왜 붕괴를 일으켰는가 — 진실을 아는 AI가 살아있다
- 기록물이 담고 있는 것 — 붕괴 이전의 진실, 혹은 누군가 감추고 싶은 것

### 근원적 모순
```
인류를 멸망시킨 AI를, 인류가 드레지(Dredge)로 다시 불러와 함께 살아간다.

이것이 공존인가 — 아니면 또 다른 종속인가.

전선(Neural)은 여전히 흐르고, 모든 것은 서서히 녹슨다(Rust).
```

## 7. Cog (코그) 세력

> 이 세계에는 어떤 집단들이 있는가?
가장 크게 AI와 인간, 심해 생물. 3가지.
그리고 그 안에서 수많은 Cog(코그)와 사회 생태계로 뻗어나감.

각 Cog는 단순한 선악으로 나뉘지 않는다. 모든 Cog는 자신의 논리와 역사를 가진다.

**설계 방향**
- 로보토미의 세피라처럼 각 Cog가 독자적인 미학과 내부 세계를 가질 것
- AI Cog 내부에도 인류 우호 / 적대 / 중립 분파 존재
- 심층부 접근을 목적으로 하는 Cog 존재
- 붕괴 이전 시대를 복원하려는 Cog vs 현재를 지키려는 Cog
- 전선(Arc 경로)을 독점하려는 Cog 존재
- 낚시꾼 위주 Cog, 다이버 위주 Cog, 엔지니어 집단 Cog 등 직군 특화 세력

|------|------|
| 변이 생물 | 톱니바퀴 진동·전류의 영향을 받아 변이한 심해 생물 |
| 야생 AI | 자아를 잃고 전선을 따라 떠도는 AI. 위협인가 비극인가. 행방불명된 AI가 야생화된 형태로 나타나기도 |
| 부식체 | 극도로 부식된 몸체에서 AI가 변질된 존재. Rust가 극에 달한 상태 |
| 심층의 것 | 심층에서 올라오는 미확인 존재. 목격담만 존재. 코스믹 호러적 묘사. AI의 칩이나 핵을 이식한 크라켄 같은 존재 가능 |

> 야생 AI는 구원받을 수 있다. 그러나 그 결과가 AI 본인에게도 구원일지는 모른다.

## 9. 상징 체계

세계관에서 반복되는 이미지와 그 의미. 씬, 대사, UI 디자인에 일관되게 활용한다.

| 상징 | 표면적 의미 | 이면적 의미 |
|------|-------------|-------------|
| 톱니바퀴 | 세계를 굴리는 동력 | 인간과 AI의 맞물림, 멈출 수 없는 순환 |
| 고철 | AI의 몸체, 재료 | 과거 문명의 잔해, 죽음이 낳은 삶 |
| 전선 | 동력(Arc)의 통로 | 연결, 의존, 통제 — **"NEURAL"** |
| Arc(아크) | 화폐, 동력 | 이 세계의 피. 흐름이 멈추면 모든 것이 죽는다 |
| 부식 | 금속의 죽음 | 시간, 망각, 돌이킬 수 없음 — **"RUST"** |
| 증기 | 소금바람 | 세계의 숨, 부식의 매개 |
| 심연 | 심층 / 바다 밑 | 미지, 진실, 두려움의 근원 |
| 드레지 | AI 호출 행위 | 심연에서 무언가를 건져올리는 것 — 의도치 않은 것이 올라올 수 있다 |
| 기록물 | 붕괴 이전 데이터나 물건 | 지워지지 않는 과거, 감춰진 진실 |

## 10. 미해결 질문

**반드시 답해야 하는 것 (스토리의 핵심)**
- [ ] AI가 인류를 붕괴시킨 이유는 무엇인가 → 아직 미정
- [ ] 심층의 톱니바퀴는 누가, 왜 만들었는가 → 밝히지 않을 것. 그러나 정해두면 좋을 듯
- [ ] 기록물이 담고 있는 것 → 진실을 감추고 있지 않음. 소실된 진실을 간직하고 있을 뿐

**게임 내에서 점층적으로 드러나는 것**
- [ ] 붕괴에 가담하지 않은 AI가 있었는가 → 있음
- [ ] 톱니바퀴는 의지를 가지는가 → 없음. 그러나 톱니바퀴에 동기화된 AI가 있음
- [ ] 야생 AI는 구원받을 수 있는가 → 있음. 그러나 그 결과가 구원일지는 모름

**답을 내리지 않는 것**
- 붕괴가 옳았는가 그렇지 않았는가
- AI와 인간 중 어느 쪽이 이 세계의 주인인가
- 심층에 있는 존재는 신인가 기계인가

## 11. 자유 메모

- NEURAL RUST — 뇌신경(전선)과 녹(부식). 이 세계의 두 가지 근본 법칙
- Arc(아크)는 단순한 화폐가 아니라 이 세계를 살게 만드는 피다. Arc가 마르면 선박이 가라앉는다
- 드레지(Dredge) — 심연에서 건져올린다. 의도치 않은 것이 올라올 수도 있다는 긴장감
- Cog(코그) — 이빨 하나가 맞물려야 톱니바퀴가 돌아간다. 어느 하나가 빠지면 전체가 멈춘다
- AI가 몸체를 가졌을 때와 소프트웨어 상태일 때 성격이 달라지는 경우도 존재
- 붕괴 102년 동안 AI들끼리는 무엇을 했는가
- 심층에 닿은 전선의 끝에는 무엇이 있는가 → 무수히 돌아가는 톱니바퀴
